C++ API Reference
=================

This section documents the C++ API for the OpenVCAD core library (``libvcad``)
and compiler library (``libvcad_compilers``). The documentation is generated
from Doxygen comments in the C++ headers.

Core Library (libvcad)
----------------------

The core library defines the foundational data structures: the implicit
``Node`` tree, geometric primitives, boolean operations, transformations,
materials, and utilities.
