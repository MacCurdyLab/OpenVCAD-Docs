``pyvcad`` module
==================

The ``pyvcad`` extension module exposes the OpenVCAD core tree, geometry
helpers, utilities, and analysis bindings. Documentation is split to mirror
the binding layout in ``core-lib/bindings/pyvcad``.

See also: :doc:`overview`, and the sections below.
