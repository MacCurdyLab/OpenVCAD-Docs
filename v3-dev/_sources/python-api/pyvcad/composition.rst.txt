Tree — composition nodes
==========================

**Composition nodes** are non-:class:`Leaf` nodes that each **own** one or more child subtrees. They describe how pieces are placed and combined in space: rigid transforms and tilings, constructive solid geometry (union, intersection, difference and bbox variants), offsets and shells.

Use them to assemble and modify designs built from :doc:`leaves` without duplicating underlying primitives.
