``pyvcad`` module
==================

The ``pyvcad`` extension module exposes the OpenVCAD core tree, attribute
model, geometry helpers, utilities, volumes, and analysis bindings.
Documentation is split to mirror the binding layout in
``core-lib/bindings/pyvcad`` and ``core-lib/bindings/pyvcad/pyvcad.cpp``.

See also: :doc:`overview`, and the sections below.
