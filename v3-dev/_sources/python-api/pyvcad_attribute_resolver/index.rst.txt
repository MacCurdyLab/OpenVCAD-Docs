``pyvcad_attribute_resolver`` module
====================================

The ``pyvcad_attribute_resolver`` package automates attribute-conversion chains
for OpenVCAD designs. It registers conversion graphs, solves for valid paths
between design-intent and machine-facing attributes, and wraps the design with
the required ``AttributeModifier`` nodes.
