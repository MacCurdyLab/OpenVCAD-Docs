``pyvcad_compilers`` module
===========================

Python bindings for classes that **compile** a prepared implicit ``Node`` into
concrete outputs (for example print-oriented slices or slicer projects). Start
with :doc:`overview` for what “compiler” means here, then :doc:`compilers` and
:doc:`utilities` for the API.
